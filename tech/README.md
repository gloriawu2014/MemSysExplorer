# Technology

This directory contains tools and resources for memory technology modeling, characterization, and fault injection.

## Modules

### 1. ArrayCharacterization (NVSim)
**Directory:** `ArrayCharacterization/`

A modified version of NVSim for circuit-level modeling of memory arrays. It estimates Area, Power, and Timing parameters for various memory technologies.
- **Documentation:** [ArrayCharacterization/README.md](ArrayCharacterization/README.md)

### 2. msxFI (Memory Fault Injection)
**Directory:** `msxFI/`

A framework for injecting faults into neural networks to evaluate their resilience to memory errors.
- **Documentation:** [msxFI/README.md](msxFI/README.md)

### 3. PINN (Device Characterization)
**Directory:** `pinn/`

A Physics-Informed Neural Network (PINN) module for characterizing RRAM devices and automatically generating NVSim configuration files.
- **Documentation:** [pinn/README.md](pinn/README.md)
